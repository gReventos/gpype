Multiplier

class gpype.Multiplier(constant=1.0, **kwargs)[source]

Bases: IONode

Multiplies the input signal by a constant factor.

The Multiplier filter applies scalar multiplication to the input signal, where each sample is multiplied by a configurable constant value. This is useful for scaling, amplification, or attenuation of signals.

Parameters:

constant (float, optional) – The multiplication factor to apply to the input signal. Defaults to 1.0 (no change).

Examples

>>> # Scale signal by factor of 2.5
>>> multiplier = Multiplier(constant=2.5)

>>> # Attenuate signal to half amplitude
>>> attenuator = Multiplier(constant=0.5)
