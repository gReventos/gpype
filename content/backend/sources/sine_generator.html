SineGenerator

class gpype.SineGenerator(sampling_rate, channel_count, frequency=1.0, amplitude=1.0, phase=0.0, **kwargs)[source]

Bases: FixedRateSource

Generates sinusoidal signals with configurable parameters.

This source generates continuous sine wave signals at a specified sampling rate. It’s commonly used for testing signal processing pipelines, creating reference signals, or simulating oscillatory phenomena. The generator maintains phase continuity across processing blocks.

Parameters:

  • sampling_rate (float) – Sampling rate for signal generation in Hz.

  • channel_count (int) – Number of output channels to generate.

  • frequency (float, optional) – Frequency of the sine wave in Hz. Must be positive. Defaults to 1.0.

  • amplitude (float, optional) – Peak amplitude of the sine wave. Must be positive. Defaults to 1.0.

  • phase (float, optional) – Initial phase offset in radians. Defaults to 0.0.

Examples

>>> # Generate 10 Hz sine wave at 250 Hz sampling rate
>>> sine_gen = SineGenerator(
...     sampling_rate=250.0,
...     channel_count=1,
...     frequency=10.0,
...     amplitude=1.0
... )

>>> # Generate multi-channel test signal with phase offset
>>> test_signal = SineGenerator(
...     sampling_rate=1000.0,
...     channel_count=8,
...     frequency=50.0,
...     amplitude=0.5,
...     phase=np.pi/4
... )

Notes

  • Phase continuity is maintained between consecutive processing blocks

  • Frequency should be much less than sampling_rate/2 (Nyquist frequency)

  • All channels generate the same sine wave signal
