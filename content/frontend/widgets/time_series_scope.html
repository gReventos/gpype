Time Series Scope

class gpype.TimeSeriesScope(time_window=10, amplitude_limit=50, **kwargs)[source]

Bases: Scope

Real-time time series visualization widget for multi-channel data.

Displays streaming time series data in a scrolling plot with configurable time window and amplitude scaling. The scope shows a moving window of recent data with automatic scrolling as new samples arrive. Multiple channels are displayed with vertical offset for clarity.

Parameters:

  • time_window (int, optional) – Length of the time window to display in seconds. Must be between 1 and 120 seconds. Defaults to 10 seconds.

  • amplitude_limit (float, optional) – Vertical scale limit for signal amplitude. Must be between 1 and 5000. This sets the +/- range for signal display. Defaults to 50.

Examples

>>> # Create scope for EEG data (10-second window, ±100µV range)
>>> eeg_scope = TimeSeriesScope(
...     time_window=10,
...     amplitude_limit=100
... )

>>> # Create scope for ECG data (5-second window, ±2mV range)
>>> ecg_scope = TimeSeriesScope(
...     time_window=5,
...     amplitude_limit=2000
... )

Notes

  • Data scrolls continuously from right to left

  • Each channel is displayed with vertical offset to prevent overlap

  • A cursor indicates the current time position

  • PAD_DURATION (0.1s) provides visual padding ahead of the cursor

  • Performance optimized for real-time display of multiple channels
